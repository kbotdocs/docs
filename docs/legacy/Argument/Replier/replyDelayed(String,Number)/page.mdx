<Section>
  일정 시간 후 메시지를 수신한 방에 메시지를 발신합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  replier.replyDelayed(msg, delay);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"msg"} type={"String"}>
        일정 시간 후 발신할 메시지의 내용
      </ParamItem>
      <ParamItem param={"delay"} type={"Number"}>
        지연할 시간(밀리초; ms)
      </ParamItem>
    </ParamList>
  </Section>
</Section>
