<Section>
  수신한 이미지를 Base64 인코딩한 값을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  imageDB.getImageBase64();
  ~~~

  <Section>
    ### 반환값

    `String` - 수신한 이미지를 Base64 인코딩한 값을 반환합니다.
  </Section>
</Section>
