<Section>
  발신자의 프로필 이미지에 대한 `android.graphics.Bitmap` 값을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  imageDB.getProfileBitmap();
  ~~~

  <Section>
    ### 반환값

    `android.graphics.Bitmap` - 발신자의 프로필 이미지에 대한 `android.graphics.Bitmap` 값을 반환합니다.
  </Section>
</Section>
