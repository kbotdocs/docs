<Section>
  앱 구동 기기의 화면이 켜져 있는지의 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Device.isScreenOn();
  ~~~

  <Section>
    ### 반환값

    `Boolean` - 앱 구동 기기의 화면이 켜져 있을 경우 `true`, 꺼져 있을 경우 `false`를 반환합니다.
  </Section>
</Section>
