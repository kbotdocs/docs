<Section>
  UI 쓰레드 상에서 특정 함수를 실행합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Api.UIThread(func);
  Api.UIThread(func, onComplete);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"func"} type={"() => Any"}>
        UI쓰레드 상에서 실행할 함수
      </ParamItem>
      <ParamItem param={"onComplete"} type={"(error, result) => Any"} optional>
        결과를 처리할 함수
        <Details summary={"매개변수"}>
          <ParamList>
            <ParamItem param={"error"} type={"java.lang.Throwable"}>
              `func` 함수 실행 도중 일어난 에러의 객체를 전달받는 인자
            </ParamItem>
            <ParamItem param={"result"} type={"Any"}>
              `func` 함수의 반환값을 전달받는 인자
            </ParamItem>
          </ParamList>
        </Details>
      </ParamItem>
    </ParamList>
  </Section>
