<Section>
  가장 최근에 수신한 이미지를 Base64 인코딩한 값을 반환합니다.

  <Noti type={"warning"}>
    해당 메소드는 안드로이드 11 이상에서 동작하지 않습니다.
  </Noti>
</Section>
<Section>
  ## 구문

  ~~~js
  Api.getLastImage();
  ~~~

  <Section>
    ### 반환값

    `String` - 가장 최근에 수신한 이미지를 Base64 인코딩한 값을 반환합니다.
  </Section>
</Section>
