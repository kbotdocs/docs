  feature: markAsReadOnID(BigInt)
---

<Section>
  특정 ID로 식별되는 방의 메시지를 읽음 처리합니다. 방에 대한 알림 읽기 세션을 찾을 수 있는지의 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Api.markAsReadOnID(roomId);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"roomId"} type={"BigInt"}>
        메시지를 읽음 처리할 방의 고유 ID
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `roomId`로 식별되는 방에 대한 알림 읽기 세션을 찾을 수 있을 경우 `true`, 찾을 수 없을 경우 `false`를 반환합니다.
  </Section>
</Section>
