<Section>
  단순 자동응답에 대하여 특정 이름을 가진 `[[전체변수]]`의 값을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Bridge.getGlobalVariable(name);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"name"} type={"String"}>
        `[[전체변수]]`의 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `name` 이름의 `[[전체변수]]`의 값을 반환합니다.
  </Section>
</Section>
