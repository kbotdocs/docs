<Section>
  `func` 함수의 실행에 소요된 시간을 반환합니다.(밀리초; ms)
</Section>
<Section>
  ## 구문

  ~~~js
  Utils.measureTimeMillis(func);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"func"} type={"() => Any?"}>
        실행 시간을 측정할 함수
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Number` - `func` 함수의 실행에 소요된 시간을 반환합니다.(밀리초; ms)
  </Section>
</Section>
