<Section>
  [`jsoup`](https://jsoup.org/)을 이용해 `url`에 동기 GET request를 보냅니다. 요청에 대한 응답을 문자열로 반환합니다. `removeTags`가 `true`일 경우 HTML
  태그를 삭제한 응답을 문자열로 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Utils.getWebText(url);
  Utils.getWebText(url, removeTags);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"url"} type={"String"}>
        GET request를 보낼 서버의 URL
      </ParamItem>
      <ParamItem param={"removeTags"} type={"Boolean"} defaultVal={"`false`"} optional>
        응답 값에 존재하는 HTML 태그를 삭제할 것인지의 여부
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - 요청에 대한 응답을 문자열로 반환합니다. `removeTags?`가 `true`일 경우 HTML 태그를 삭제한 응답을 문자열로 반환합니다.
  </Section>
</Section>
