<Section>
  [`jsoup`](https://jsoup.org/)을 이용해 `url`에 동기 GET request를 보냅니다. 요청에 대한 응답을 `org.jsoup.nodes.Document` 값으로 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Utils.parse(url);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"url"} type={"String"}>
        GET request를 보낼 서버의 URL
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `org.jsoup.nodes.Document` - 요청에 대한 응답을 `org.jsoup.nodes.Document` 값으로 반환합니다.
  </Section>
</Section>
