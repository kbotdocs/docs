<Section>
  `onCreate`는 스크립트 액티비티의
  [`onCreate`](https://developer.android.com/reference/android/app/Activity#onCreate(android.os.Bundle))가 호출되면 동작하는 이벤트 리스너입니다.
</Section>
<Section>
  ## 구문

  ~~~js
  function onCreate(savedInstanceState, activity) {

  }
  ~~~

  <Section>
    ### 인자

    <ParamList>
      <ParamItem param={"savedInstanceState"} type={"android.os.Bundle"}>
        스크립트 액티비티의 `onCreate`의 인자 `savedInstanceState`입니다.
      </ParamItem>
      <ParamItem param={"activity"} type={"android.app.Activity"}>
        해당 이벤트가 발생했을 때의 액티비티입니다.
      </ParamItem>
    </ParamList>
  </Section>
</Section>
