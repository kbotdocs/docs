<Section>
  # clearInterval()

  [`setInterval()`](/reference/legacy/Function/setInterval)으로 생성한 반복 작업을 취소합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  clearInterval(intervalID);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"intervalID"} type={"Number"}>
        취소할 반복 작업의 ID. 해당 값은 [`setInterval()`](/reference/legacy/Function/setInterval) 호출 시 반환된 값입니다.
      </ParamItem>
    </ParamList>
  </Section>
</Section>
