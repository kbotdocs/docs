<Section>
  `console` 객체는 앱 내 스크립트 로그에 로그를 기록할 수 있게 하는 객체입니다.

  <Noti type={"info"}>
    `console` 객체를 사용하기 위해 스크립트를 생성할 때 **Console API를 사용합니다 (실험적)** 항목을 선택해야 합니다.
  </Noti>
</Section>
