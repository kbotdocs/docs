<Section>
  ## console.log()

  정보 로그를 작성합니다. 해당 메소드의 동작은 [`console.info`](/reference/legacy/Object/console/info)`(str, false)`와 같습니다.
</Section>
<Section>
  ## 구문

  ~~~js
  console.log(str);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"str"} type={"String"}>
        작성할 정보 로그의 내용
      </ParamItem>
    </ParamList>
  </Section>
</Section>
