  특정 레이블의 호출 횟수를 초기화합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  console.countReset();
  console.countReset(label);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"label"} type={"String"} defaultVal={"`\"default\"`"} optional>
        호출 횟수를 초기화할 레이블
      </ParamItem>
    </ParamList>
  </Section>
