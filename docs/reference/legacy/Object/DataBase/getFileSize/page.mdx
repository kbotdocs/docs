<Section>
  `fileName` 파일의 용량(Byte)을 반환합니다. 파일이 존재하지 않거나 디렉토리일 경우 `-1`을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  DataBase.exists(fileName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"fileName"} type={"String"}>
        용량을 확인할 파일의 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Number` - `fileName` 파일의 용량(Byte)을 반환합니다. 파일이 존재하지 않을 경우 `-1`을 반환합니다.
  </Section>
</Section>
