<Section>
  특정 지역 또는 특정 행정구역코드로 식별되는 지역의 기상 정보를 반환합니다.
  행정구역코드는 [`Utils.getZoneIdByName()`](/reference/legacy/Object/Utils/getZoneIdByName) 메소드를 사용하거나 [여기](https://www.code.go.kr/stdcode/regCodeL.do)에서 구할 수 있습니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Utils.getWeatherJSON(locationOrZoneId);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"locationOrZoneId"} type={"String"}>
        기상 정보를 조회할 지역 또는 지역의 행정구역코드
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `locationOrZoneId`로 식별되는 지역의 기상 정보를 JSON 형태의 문자열로 반환합니다.
  </Section>
</Section>
