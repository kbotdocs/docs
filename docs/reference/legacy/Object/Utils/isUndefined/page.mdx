<Section>
  `value`가 `undefined`인지의 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Utils.isUndefined(value);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"value"} type={"Any?"}>
        `undefined`인지 확인할 값
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `value`가 `undefined`이면 `true`, 아니면 `false`를 반환합니다.
  </Section>
</Section>
