<Section>
  앱의 Context(컨텍스트)를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Api.getContext();
  ~~~

  <Section>
    ### 반환값

    `android.content.Context` - 앱의 Context(컨텍스트)를 반환합니다.
  </Section>
