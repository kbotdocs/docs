<Section>
  `path` 경로가 존재하는지의 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  FileStream.exists(path);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"path"} type={"String"}>
        존재 여부를 확인할 경로
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `path` 경로가 존재할 경우 `true`, 존재하지 않을 경우 `false`를 반환합니다.
  </Section>
</Section>
