<Section>
  `path` 경로에서 파일의 이름을 반환합니다. 경로의 존재 여부와 디렉토리일 경우를 고려하지 않습니다.
</Section>
<Section>
  ## 구문

  ~~~js
  FileStream.getFileName(path);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"path"} type={"String"}>
        파일의 이름을 확인할 경로
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `path` 경로에서 파일의 이름을 반환합니다.
  </Section>
</Section>
