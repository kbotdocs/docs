<Section>
  단순 자동응답에 대하여 특정 이름을 가진 `[[전체변수]]`에 값을 저장합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Bridge.setGlobalVariable(name, value);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"name"} type={"String"}>
        값을 저장할 `[[전체변수]]`의 이름
      </ParamItem>
      <ParamItem param={"value"} type={"String"}>
        저장할 값
      </ParamItem>
    </ParamList>
  </Section>
</Section>
