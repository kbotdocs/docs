<Section>
  `Bridge` 객체는 다른 스크립트의 전역 스코프에 접근하기 위한 객체입니다. 채팅 자동응답 봇에서는 단순 자동응답 간의 스코프 교류를 위한 메소드도 제공합니다.
</Section>
