<Section>
  발신자의 프로필 이미지를 SHA 인코딩한 값을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  imageDB.getProfileSHA();
  ~~~

  <Section>
    ### 반환값

    `String` - 발신자의 프로필 이미지를 SHA 인코딩한 값을 반환합니다.
  </Section>
</Section>
