<Section>
  수신한 이미지를 SHA 인코딩한 값을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  imageDB.getImageSHA();
  ~~~

  <Section>
    ### 반환값

    `String` - 수신한 이미지를 SHA 인코딩한 값을 반환합니다.
  </Section>
</Section>
