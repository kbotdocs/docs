<Section>
  `ImageDB` 객체는 이미지에 대한 기능을 제공하는 객체입니다. 해당 객체는 직접 호출할 수 없습니다. [`response`](/reference/legacy/EventListener/response)의 인자로 주어집니다.
</Section>
