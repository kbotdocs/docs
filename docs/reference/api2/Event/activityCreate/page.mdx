<Section>
  `activityCreate` 이벤트는 스크립트 액티비티의
  [`onCreate`](https://developer.android.com/reference/android/app/Activity#onCreate(android.os.Bundle))가 호출되면 발생하는
  이벤트입니다.
</Section>
<Section>
  ## 구문

  [`Bot.addListener()`](/reference/api2/Object/Bot/addListener)를 이용해 이벤트에 대한 리스너를 추가할 수 있습니다.
  [`Event.Activity.CREATE`](/reference/api2/Object/Event/Activity/CREATE)를 이용해 이벤트 이름을 전달할 수도 있습니다.
  ~~~js
  bot.addListener("activityCreate", (activity) => {});
  // 또는
  bot.addListener(Event.Activity.CREATE, (activity) => {});
  ~~~

  <Section>
    ### 인자

    <ParamList>
      <ParamItem param={"activity"} type={"android.app.Activity"}>
        해당 이벤트가 발생했을 때의 액티비티입니다.
      </ParamItem>
    </ParamList>
  </Section>
</Section>
