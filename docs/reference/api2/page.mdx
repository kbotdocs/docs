<Section>
  API2는 최근에 공개된 내장 API로, 레거시 API보다 더 체계적인 구조로 되어 있습니다. 카카오톡봇의 지원은 API2를 중심으로 이루어질 예정입니다.
</Section>

  <Table>
    <Thead>
      <Tr>
