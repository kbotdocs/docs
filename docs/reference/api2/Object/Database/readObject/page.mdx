<Section>
  `fileName` 이름의 파일의 내용을 `Object` 형식으로 반환합니다. 파일 내용의 형식이 `JSON`일 경우에만 가능합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Database.readObject(fileName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"fileName"} type={"String"}>
        내용을 `Object` 형식으로 읽을 파일 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Object` - `fileName` 이름의 파일의 내용을 `Object` 형식으로 반환합니다.
  </Section>
</Section>
