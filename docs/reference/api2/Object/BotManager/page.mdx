<Section>
  `BotManager` 객체는 `Bot` 객체를 관리/보조해주는 객체입니다. 개별 `Bot` 객체를 관리할 수도 있지만, 모든 `Bot` 객체에 대해 동작을 수행할 수도 있습니다.
</Section>
