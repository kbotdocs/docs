<Section>
  메시지 전송이 가능한 방 이름을 배열로 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  BotManager.getRooms();
  BotManager.getRooms(packageName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"packageName"} type={"String"} optional>
        메시지가 발신된 방들의 메신저 앱 패키지명
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String[]` - `packageName` 인자가 존재할 경우 `packageName`의 메신저 앱에 대해 메시지 전송이 가능한 방 이름이 담긴 배열을 반환합니다.
    `packageName` 인자가 존재하지 않을 경우 모든 메신저 앱에 대해 메시지 전송이 가능한 방 이름이 담긴 배열을 반환합니다.
  </Section>
</Section>
