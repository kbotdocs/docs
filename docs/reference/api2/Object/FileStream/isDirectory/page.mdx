<Section>
  `path` 경로가 디렉토리인지의 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  FileStream.isDirectory(path);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"path"} type={"String"}>
        디렉토리인지 확인할 경로
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `path`가 디렉토리일 경우 `true`, 디렉토리가 아닐 경우 `false`를 반환합니다.
  </Section>
</Section>
