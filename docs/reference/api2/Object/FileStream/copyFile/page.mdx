<Section>
  `path1` 경로의 파일을 `path2` 경로에 복사합니다. 파일 복사의 성공 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  FileStream.copyFile(path1, path2);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"path1"} type={"String"}>
        복사할 원본 파일의 경로
      </ParamItem>
      <ParamItem param={"path2"} type={"String"}>
        복사본의 경로
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - 파일 복사에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
  </Section>
</Section>
