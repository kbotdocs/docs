<Section>
  특정 방에 메시지를 보냅니다. 메시지를 보낼 방에 대한 세션의 존재 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  bot.send(room, msg);
  bot.send(room, msg, packageName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"room"} type={"any"}>
        메시지를 보낼 방의 식별자. 식별자는 방의 이름 또는 방의 ID를 사용할 수 있습니다.
      </ParamItem>
      <ParamItem param={"msg"} type={"String"}>
        보낼 메시지의 내용
      </ParamItem>
      <ParamItem param={"packageName"} type={"String"} optional>
        메시지를 보낼 메신저 앱의 패키지명
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `room` 방에 대한 세션이 존재할 경우 `true`, 존재하지 않을 경우 `false`를 반환합니다.
  </Section>
</Section>
