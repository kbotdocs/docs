<Section>
  특정 이벤트에 대한 이벤트 리스너를 리스너 리스트의 마지막에 추가합니다. 해당 메소드의 동작은 [`Bot.addListener()`](/reference/api2/Object/Bot/addListener)와 동일합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  bot.on(eventName, listener);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"eventName"} type={"String"}>
        리스너를 추가할 이벤트명
      </ParamItem>
      <ParamItem param={"listener"} type={"(arg1, arg2, ..., argN) => Any"}>
        추가할 이벤트 리스너
        <Details summary={"매개변수"}>
          <ParamList>
            <ParamItem param={"arg1, arg2, ..., argN"} type={"Any"}>
              `eventName` 이벤트에 의해 주어지는 인자에 대한 매개변수입니다. API2의 이벤트는 [여기](/api2#이벤트)를 참고하세요.
            </ParamItem>
          </ParamList>
        </Details>
      </ParamItem>
    </ParamList>
  </Section>
</Section>
