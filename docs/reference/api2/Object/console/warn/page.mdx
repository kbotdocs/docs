<Section>
  주의 로그를 작성합니다. 여러 개의 인자를 전달할 수 있으며, 각 인자는 공백으로 구분되어 작성됩니다. 해당 메소드의 동작은 [`console.info()`](/reference/api2/Object/console/info)와 같습니다.
</Section>
<Section>
  ## 구문

  ~~~js
  console.warn();
  console.warn(arg1);
  console.warn(arg1, arg2, /* ..., */ argN);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"...args"} type={"any"} optional>
        작성할 정보 로그의 내용 
      </ParamItem>
    </ParamList>
  </Section>
</Section>
