<Section>
  `Broadcast` 객체는 다른 스크립트 간의 데이터 교류를 지원하는 객체입니다. [`Broadcast.send()`](/reference/api2/Object/Broadcast/send)를 통해 전송된 브로드캐스트는
  [`Broadcast.register()`](/reference/api2/Object/Broadcast/register)를 이용해 모든 스크립트에서 감지하고 접근할 수 있습니다.

  <Noti type="info">
    Android의 [브로드캐스트](https://developer.android.com/develop/background-work/background-tasks/broadcasts?hl=ko)와 다릅니다.
  </Noti>
</Section>
