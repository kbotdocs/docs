  ## Security.getDesKey()

  (DES 보조키를 생성하여 반환합니다.) 해당 메소드는 구현되지 않았습니다. 호출 시 `java.lang.RuntimeException`을 throw합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Security.getDesKey([2arguments]);
  ~~~

  <Section>
    ### 매개변수

    최대 2개의 매개변수를 받습니다.
  </Section>
  <Section>
    ### 반환값

    (`String`) - (생성한 DES 보조키를 반환합니다.)
    `java.lang.RuntimeException`을 throw합니다.
  </Section>
</Section>
