<Section>
    # App.runOnUiThread()

    UI 스레드 상에서 특정 함수를 실행합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    App.runOnUiThread(task);
    App.runOnUiThread(task, onComplete);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"task"} type={"() => Any"}>
                UI 스레드 상에서 실행할 함수
            </ParamItem>
            <ParamItem param={"onComplete"} type={"(error, result) => Any"} optional>
                결과를 처리할 함수
            </ParamItem>
            <ParamList>
                <ParamItem param={"error"} type={"Rhino: org.mozilla.javascript.JavaScriptException, GraalJS: org.graalvm.polyglot.PolyglotException"}>
                    `task` 함수 실행 도중 일어난 에러의 객체를 전달받는 인자
                </ParamItem>
                <ParamItem param={"result"} type={"Any"}>
                    `task` 함수의 반환값을 전달받는 인자
                </ParamItem>
            </ParamList>
        </ParamList>
    </Section>
</Section>
