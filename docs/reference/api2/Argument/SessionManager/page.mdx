<Section>
  `SessionManager` 객체는 방 세션을 수동으로 등록할 수 있게 해주는 객체입니다. 해당 객체는 직접 호출할 수 없습니다.
  [`notificationPosted`](/reference/api2/Event/notificationPosted) 이벤트의 인자로 주어집니다.
</Section>
