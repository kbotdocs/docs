<Section>
  설정한 값들로 알림을 생성하고 생성된 알림을 제어할 수 있는 [`Notification`](/reference/starlight/Object/Notification) 객체를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  NotificationBuilder.build();
  ~~~

  <Section>
    ### 반환값

    `Notification` - 생성된 알림을 제어할 수 있는 [`Notification`](/reference/starlight/Object/Notification) 객체를 반환합니다.
  </Section>
</Section>
