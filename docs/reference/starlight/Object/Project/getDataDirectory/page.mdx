---
title: Project.getDataDirectory()
category: StarLight
---

<Section>
  프로젝트의 데이터(데이터베이스 파일, 유저 데이터 등)가 저장된 디렉터리에 대한 `java.io.File`을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Project.getDataDirectory();
  ~~~

  <Section>
    ### 반환값

    `java.io.File` - 프로젝트의 데이터가 저장된 디렉터리에 대한 `java.io.File`을 반환합니다.
  </Section>
</Section>
