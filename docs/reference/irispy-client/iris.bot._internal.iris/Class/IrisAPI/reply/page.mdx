<Section>
  특정 방에 메시지를 보냅니다.
</Section>
<Section>
  ## 구문

  ~~~py
  irisAPI.reply(room_id, msg)
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"room_id"} type={"int"}>
        메시지를 보낼 방의 id
      </ParamItem>
      <ParamItem param={"msg"} type={"str"}>
        보낼 메시지의 내용
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `dict` - 메시지 전송에 대한 결과를 반환합니다. 자세한 내용은 [여기](/reference/iris/Endpoint/reply)를 참조하세요.
  </Section>
</Section>
