title: Bot(str, *, int=None)
category: irispy-client

<Section>
  Iris를 구동할 URL과 이벤트를 처리할 이벤트 수에서 새 `Bot` 인스턴스를 생성합니다.
</Section>
<Section>
  ## 구문

  ~~~py
  Bot(iris_url)
  Bot(iris_url, max_workers=_max_workers)
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"iris_url"} type={"str"}>
        Iris가 사용할 서버의 URL
      </ParamItem>
      <ParamItem param={"max_workers"} type={"int"} defaultVal={"`None`"} optional>
        이벤트 처리에 사용할 최대 스레드 수
      </ParamItem>
    </ParamList>
  </Section>
</Section>
