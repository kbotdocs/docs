---
title: PyKV.get()
category: irispy-client
---

<Section>
  데이터베이스에서 특정 키에 대응하는 값을 반환합니다. 키에 대응하는 값이 존재하지 않을 경우 `False`를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~py
  pyKV.get(key)
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"key"} type={"str"}>
        가져올 값에 대응하는 키
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    *[`Any`](https://docs.python.org/3/library/typing.html#typing.Any)* - 특정 키에 대응하는 값을 반환합니다. 키에 대응하는 값이 존재하지 않을 경우 `False`를 반환합니다.
  </Section>
</Section>
