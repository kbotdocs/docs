<Section>
  답장 대상 메시지에 대한 `ChatContext`를 반환합니다. 목표 채팅이 존재하지 않을 경우 `None`을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~py
  chatContext.get_source()
  ~~~

  <Section>
    ### 반환값

    [`iris.bot.models.ChatContext`](/reference/irispy-client/iris.bot.models/Class/ChatContext) - 답장 대상 메시지에 대한 `ChatContext`를 반환합니다. 목표 채팅이 존재하지 않을 경우 `None`을 반환합니다.
  </Section>
</Section>
