<Section>
  `activityPause` 이벤트는 스크립트 액티비티의 [`onPause`](https://developer.android.com/reference/android/app/Activity#onPause())가
  호출되면 발생하는 이벤트입니다.
</Section>
<Section>
  ## 구문

  [`Bot.addListener()`](/api2/Object/Bot/addListener)를 이용해 이벤트에 대한 리스너를 추가할 수 있습니다.
  [`Event.Activity.PAUSE`](/api2/Object/Event/Activity/PAUSE)를 이용해 이벤트 이름을 전달할 수도 있습니다.
  ~~~js
  bot.addListener("activityPause", (activity) => {});
  // 또는
  bot.addListener(Event.Activity.PAUSE, (activity) => {});
  ~~~

  <Section>
    ### 인자

    <ParamList>
      <ParamItem param={"activity"} type={"android.app.Activity"}>
        해당 이벤트가 발생했을 때의 액티비티입니다.
      </ParamItem>
    </ParamList>
  </Section>
</Section>
