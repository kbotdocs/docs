<Section>
  `command` 이벤트는 [`Bot.setCommandPrefix()`](/api2/Object/Bot/setCommandPrefix)에 의해 정해진 접두어로 시작하는 명령어를 수신할 때 발생하는 이벤트입니다.
</Section>
<Section>
  ## 구문

  [`Bot.addListener()`](/api2/Object/Bot/addListener)를 이용해 이벤트에 대한 리스너를 추가할 수 있습니다. [`Event.COMMAND`](/api2/Object/Event/COMMAND)를 이용해
  이벤트 이름을 전달할 수도 있습니다.
  ~~~js
  bot.addListener("command", (msg) => {});
  // 또는
  bot.addListener(Event.COMMAND, (msg) => {});
  ~~~

  <Section>
    ### 인자

    <ParamList>
      <ParamItem param={"msg"} type={"Command"}>
        발신된 명령어에 대한 정보를 담고 있는 [`Command`](/api2/Argument/Command) 객체입니다.
      </ParamItem>
    </ParamList>
  </Section>
</Section>
