<Section>
  `value`를 SEED 복호화한 값을 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Security.seedDecode(key, value);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"key"} type={"String"}>
        SEED 복호화를 위한 키
      </ParamItem>
      <ParamItem param={"value"} type={"String"}>
        복호화할 값
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `value`를 SEED 복호화한 값을 반환합니다.
  </Section>
</Section>
