<Section>
  `Bot`객체에 대응하는 스크립트의 활성화 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  bot.getPower();
  ~~~

  <Section>
    ### 반환값

    `Boolean` - 스크립트가 활성화 상태일 경우 `true`, 비활성화 상태일 경우 `false`를 반환합니다.
  </Section>
</Section>
