<Section>
    # App.runOnUiThread()

    UI쓰레드 상에서 특정 함수를 실행합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    App.runOnUiThread(task, onComplete);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"task"} type={"() => Any"}>
                UI쓰레드 상에서 실행할 함수
            </ParamItem>
            <ParamItem param={"onComplete"} type={"(error, result) => Any"}>
                결과를 처리할 함수
                <Details summary={"매개변수"}>
                    <ParamList>
                        <ParamItem param={"error"} type={"org.mozilla.javascript.JavaScriptException"}>
                            `task` 함수 실행 도중 일어난 에러의 객체를 전달받는 인자
                        </ParamItem>
                        <ParamItem param={"result"} type={"Any"}>
                            `task` 함수의 반환값을 전달받는 인자
                        </ParamItem>
                    </ParamList>
                </Details>
            </ParamItem>
        </ParamList>
    </Section>
</Section>
