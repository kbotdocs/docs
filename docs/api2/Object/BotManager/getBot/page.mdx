<Section>
  `botName` 이름의 `Bot` 객체를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  BotManager.getBot(botName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"botName"} type={"String"}>
        반환받을 `Bot` 객체의 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `botName` 이름의 `Bot` 객체를 반환합니다.
  </Section>
</Section>
