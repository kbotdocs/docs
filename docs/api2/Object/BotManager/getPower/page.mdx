<Section>
  `botName` 이름의 `Bot` 객체의 활성화 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  BotManager.getPower(botName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"botName"} type={"String"}>
        활성화 여부를 확인할 `Bot` 객체의 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `botName`의 `Bot`이 활성화 상태일 경우 `true`, 비활성화 상태일 경우 `false`를 반환합니다.
  </Section>
</Section>
