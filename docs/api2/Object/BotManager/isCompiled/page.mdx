<Section>
  `botName` 이름의 `Bot` 객체(스크립트)의 컴파일 완료 여부를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  BotManager.isCompiled(botName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"botName"} type={"String"}>
        컴파일 완료 여부를 확인할 `Bot` 객체의 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Boolean` - `botName` 이름의 `Bot` 객체(스크립트)의 컴파일이 완료된 경우 `true`, 완료되지 않은 경우 `false`를 반환합니다.
  </Section>
</Section>
