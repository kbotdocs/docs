<Section>
  `botName` 이름의 `Bot` 객체를 컴파일 전 상태로 전환(언로드)합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  BotManager.unload(botName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"botName"} type={"String"}>
        언로드할 `Bot` 객체의 이름
      </ParamItem>
    </ParamList>
  </Section>
</Section>
