<Section>
  `botName` 이름의 `Bot` 객체의 활성화 여부를 제어합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  BotManager.setPower(botName, power);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"botName"} type={"String"}>
        활성화 여부를 제어할 `Bot` 객체의 이름
      </ParamItem>
      <ParamItem param={"power"} type={"Boolean"}>
        `Bot` 객체의 활성화 여부
      </ParamItem>
    </ParamList>
  </Section>
</Section>
