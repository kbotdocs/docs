<Section>
  앱 데이터에 `key: boolean` (`String: Boolean` 형식) 데이터를 씁니다.
</Section>
<Section>
  ## 구문

  ~~~js
  AppData.putBoolean(key, boolean);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"key"} type={"String"}>
        쓸 앱 데이터의 키
      </ParamItem>
      <ParamItem param={"boolean"} type={"Boolean"}>
        `key`에 대한 `Boolean`형 데이터
      </ParamItem>
    </ParamList>
  </Section>
</Section>
