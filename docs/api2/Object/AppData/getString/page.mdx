<Section>
    # AppData.getString()

    `key`에 대응하는 `String`형 데이터를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    AppData.getString(key);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"key"} type={"String"}>
                불러올 `String`형 데이터에 대응하는 키
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `String` - `key`에 대응하는 `String`형 데이터를 반환합니다.
    </Section>
</Section>
