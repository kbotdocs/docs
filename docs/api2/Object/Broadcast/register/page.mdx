<Section>
  `broadcastName`를 이름으로 하는 브로드캐스트에 대한 리스너를 추가합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Broadcast.register(broadcastName, task);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"broadcastName"} type={"String"}>
        리스너를 추가할 브로드캐스트 이름
      </ParamItem>
      <ParamItem param={"task"} type={"(value) => Any"}>
        리스너로 추가할 함수
        <Details summary={"매개변수"}>
          <ParamList>
            <ParamItem param={"value"} type={"Any"}>
              브로드캐스트을 통해 송출된 값
            </ParamItem>
          </ParamList>
        </Details>
      </ParamItem>
    </ParamList>
  </Section>
</Section>
