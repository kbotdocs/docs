<Section>
  `broadcastName`를 이름으로 하는 브로드캐스트에 대한 특정 리스너를 제거합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Broadcast.unregister(broadcastName, task);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"broadcastName"} type={"String"}>
        리스너를 제거할 브로드캐스트 이름
      </ParamItem>
      <ParamItem param={"task"} type={"(value) => Any"}>
        제거할 리스너
        <Details summary={"매개변수"}>
          <ParamList>
            <ParamItem param={"value"} type={"Any"}>
              브로드캐스트을 통해 송출된 값
            </ParamItem>
          </ParamList>
        </Details>
      </ParamItem>
    </ParamList>
  </Section>
</Section>
