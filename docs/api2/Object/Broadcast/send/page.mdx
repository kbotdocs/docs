<Section>
  모든 스크립트를 대상으로 특정 값을 가진 브로드캐스트를 전송합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Broadcast.send(broadcastName, value);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"broadcastName"} type={"String"}>
        전송할 브로드캐스트 이름
      </ParamItem>
      <ParamItem param={"value"} type={"Any"}>
        브로드캐스트로 송출할 값
      </ParamItem>
    </ParamList>
  </Section>
</Section>
