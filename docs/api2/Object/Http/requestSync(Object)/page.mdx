<Section>
  [`jsoup`](https://jsoup.org/)을 이용해 옵션에 따라 동기 request를 보냅니다. 요청에 대한 응답을 `org.jsoup.nodes.Document` 형식으로 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Http.requestSync(option);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"option"} type={"Object"}>
        보낼 요청의 옵션
        <Details summary={"`option` 정보"}>
          ~~~json
          {
            "url": String,
            "timeout": Number,
            "method": String,
            "headers": { /* Header */}
          }
          ~~~
          <ParamList>
            <ParamItem param={"url"} type={"String"}>
              request를 보낼 URL
            </ParamItem>
            <ParamItem param={"timeout"} type={"Number"} defaultVal={"`3000`"} optional>
              request 타임아웃 (밀리초; ms)
            </ParamItem>
            <ParamItem param={"method"} type={"String"} defaultVal={"`\"GET\"`"} optional>
              request 메소드
            </ParamItem>
            <ParamItem param={"headers"} type={"Object"} defaultVal={"`null`"} optional>
              request 헤더
            </ParamItem>
          </ParamList>
        </Details>
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    [`org.jsoup.nodes.Document`](https://jsoup.org/apidocs/org/jsoup/nodes/Document.html) - 요청에 대한 응답을
    `org.jsoup.nodes.Document` 형식으로 반환합니다.
  </Section>
</Section>
