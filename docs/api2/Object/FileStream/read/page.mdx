<Section>
  `path` 경로의 파일 내용을 문자열로 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  FileStream.read(path);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"path"} type={"String"}>
        내용을 문자열로 읽을 파일의 경로
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `path` 경로의 파일 내용을 반환합니다.
  </Section>
</Section>
