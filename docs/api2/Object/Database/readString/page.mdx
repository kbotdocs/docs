<Section>
  `fileName` 이름의 파일의 내용을 문자열로 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Database.readString(fileName);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"fileName"} type={"String"}>
        내용을 문자열로 읽을 파일 이름
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `String` - `fileName` 이름의 파일의 내용을 반환합니다.
  </Section>
</Section>
