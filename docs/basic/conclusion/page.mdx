  여기까지 모든 코스를 성실히 밟으신 분들께 축하의 말씀을 드립니다! 객체 기초 편을 마지막으로 Rhino JavaScript를 사용한 카카오톡봇 기초 자습서를 마칩니다.

  기본적인 JavaScript에 익숙해지셨다면 중급 자습서를 시작해보세요. 기초 자습서에서 소개한 객체는 JavaScript의 객체의 전부가 아닙니다. 객체를 다루는 방법론은 훨씬 더 방대하거든요.

  중급 자습서에서는 더 본격적인 JavaScript 객체, 다양한 반복 처리 등 한층 더 심화된 JavaScript와 Java Class의 접근과 같은 Rhino 엔진 기반의 카카오톡봇을 위한 더 많은 정보를 소개할 예정입니다.
  JavaScript와 카카오톡봇에 본격적으로 파고들고 싶다면 중급 자습서를 추천합니다.
