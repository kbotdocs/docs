---
title: Plugins.ofId()
category: StarLight
---

<Section>
  특정 ID로 식별되는 플러그인에 대한 [`Plugin`](/starlight/Object/Plugin) 객체를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Plugins.ofId(id);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"id"} type={"String"}>
        플러그인의 고유 ID
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Plugin` - `id`로 식별되는 플러그인에 대한 `Plugin` 객체를 반환합니다.
  </Section>
</Section>
