---
title: Languages.ofId()
category: StarLight
---

<Section>
  특정 ID로 식별되는 프로그래밍 언어 구현체에 대한 [`Language`](/starlight/Object/Language) 객체를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Languages.ofId(id);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"id"} type={"String"}>
        언어 구현체의 ID
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `Language` - `id`로 식별되는 프로그래밍 언어 구현체에 대한 `Language` 객체를 반환합니다.
  </Section>
</Section>
