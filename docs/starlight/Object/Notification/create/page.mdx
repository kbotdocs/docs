---
title: Notification.create()
category: StarLight
---

<Section>
  특정 ID의 알림을 생성하는 [`NotificationBuilder`](/starlight/Object/NotificationBuilder) 객체를 반환합니다.
</Section>
<Section>
  ## 구문

  ~~~js
  Notification.create(id);
  ~~~

  <Section>
    ### 매개변수

    <ParamList>
      <ParamItem param={"id"} type={"Number"}>
        생성될 알림의 ID (정수)
      </ParamItem>
    </ParamList>
  </Section>
  <Section>
    ### 반환값

    `NotificationBuilder` - `id`를 ID로 가지는 알림을 생성하는 `NotificationBuilder` 객체를 반환합니다.
  </Section>
</Section>
